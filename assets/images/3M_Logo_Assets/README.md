# 3M Logo Asset Pack

Separated from the single `3m_logo3.ai` sheet into 17 individual, ready-to-use logo assets — each with a version for **light backgrounds** and a version for **dark backgrounds**, so they drop straight into web pages or video edits without extra prep.

## Folders

- **01_Video_HighRes/** — full resolution (900 dpi-equivalent, rendered directly from the vector source). Use these in Premiere/After Effects/DaVinci, on large screens, or anywhere you might scale the logo up.
- **02_Web_Optimized/** — same assets downscaled to a 1600px max dimension and re-compressed. Use these on the website — much smaller file size, loads faster, still sharp at normal display sizes.

All files are transparent PNGs (no background), so they can be placed over any color, photo, or video footage.

## File naming

Each asset has two files:

- `{name}_light.png` — original colors, for light/white backgrounds.
- `{name}_dark.png` — a white version of the same logo, for dark backgrounds and dark video scenes.

One exception: **`scotchgard-protector-badge_universal.png`** — this badge has a solid white card built into the artwork itself, so it already works on any background color as-is. There's no separate dark variant for it (a synthetic white version would just turn it into a solid white blob, since the badge is already white).

Two of the Scotchgard cards (`scotchgard-pro-200gloss-card` and `scotchgard-pro-200matte-card`) contain that same white-carded diamond badge embedded above their text. Their `_dark.png` versions were built carefully so the diamond badge stays untouched (it's already dark-bg-safe) while only the plain text below it converts to white — otherwise the same white-blob problem would happen.

## Usage tips

- On the website: use the `02_Web_Optimized` light version on white/light sections, and the dark version on any dark-themed section, hero image overlay, or footer.
- In video: use `01_Video_HighRes` so the logo stays crisp even if you scale it up in the timeline; pick light or dark to match the footage/background behind it.
- Colors (the red 3M mark, the blue Crystalline wordmark) are preserved in both light and dark versions — only the black line art/text switches to white.
